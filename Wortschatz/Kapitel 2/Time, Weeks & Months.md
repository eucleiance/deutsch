#### Time

| Die Zeit       | Time  |
| -------------- | ----- |
| das Jahr, -e   | year  |
| der Monat, -e  | month |
| die Stunde, -n | hour  |
| der Tag, -e    | day   |
| die Zeit, -en  | time  |
